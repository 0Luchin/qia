# Instalación de QIA v2

Bienvenida a QIA. Para empezar, abre tu terminal de WSL/Linux y ejecuta estos dos pasos:

### 1. Preparar el sistema
```bash
sudo apt update
sudo apt install -y make wget python3
```

### 2. Instalar QIA
```bash
make install
```

¡Listo! Ya puedes escribir `qia status` en tu terminal para verificar que todo funciona.
