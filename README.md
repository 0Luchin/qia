# qia

Asistente local de terminal para Linux/WSL usando Ollama + modelos locales.

## Comandos

- q: consulta rápida
- qdo: genera o propone comandos de terminal
- qmodel: gestiona modelo activo
- qprofile: gestiona perfil/prompt activo
- qia.py: motor principal

## Backend actual

Ollama local en:

http://127.0.0.1:11434

## Configuración

La configuración original vive en:

~/.config/qia/

En este repo se guarda una copia en:

config/

## Nota

Los modelos de Ollama no se guardan en este repositorio porque pesan varios GB.
Ver modelos_ollama_actuales.txt.
